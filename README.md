# **CyberAwarenessChatbot**



https://github.com/ST10207887/IIEWFL-PROG6221-POE-PART1-ST10207887.git



**Overview**

CyberAwarenessChatbot is an interactive console application designed to raise awareness about cybersecurity practices. It guides users through common scenarios, provides helpful tips, and reinforces safe online behaviour in a conversational format.



**Features**

\- Interactive chatbot experience

\- Guidance on safe password practices

\- Tips for avoiding phishing and social engineering

\- Awareness of malware and ransomware risks

\- Simple, user‑friendly console interface



**Setup Instructions**

**Prerequisites**

\- .NET 8 SDK (dotnet.microsoft.com in Bing) installed

\- GitHub account and Git installed locally



**Continuous integration**

This project uses GitHub Actions for CI/CD:

\- Automatically builds the project on every push and pull request

\- Restores dependencies, compiles, and runs tests
